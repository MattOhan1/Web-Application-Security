🧰 Web Security Tools

1. 🕷️ OWASP ZAP (Zed Attack Proxy)

A free, open-source security scanner used to find vulnerabilities in web applications during development and testing.
🔗 Visit OWASP ZAP

2. 🦠 Burp Suite

A powerful platform used for web app security testing. It includes tools for scanning, crawling, and intercepting traffic.
🔗 Visit Burp Suite

3. 🔍 Nikto

A command-line tool that scans web servers for dangerous files, outdated software, and other common security issues.
🔗 Visit Nikto

4. 🧪 Snyk

Snyk helps find and fix vulnerabilities in open source dependencies, containers, and code — often used in DevSecOps.
🔗 Visit Snyk

5. 🧰 Nmap

A network scanning tool used to discover hosts, services, and security issues across networks. Not web-specific, but still helpful for overall security.
🔗 Visit Nmap
